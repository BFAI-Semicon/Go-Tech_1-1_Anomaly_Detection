# 半導体製造における光学画像・外観検査の異常検知 文献調査計画

## 1. 目的

- 半導体製造工程（ウェハ／ダイ／パッケージ等）の光学撮像画像による不良検知に関する最新アルゴリズムを体系的に文献調査し、採用候補を絞り込む。
- 重点観点：産業適用性（データ要件・速度報告・ロバスト性記述・実運用事例）、再現性（公開実装・依存関係）、ライセンス条件、今後の実験計画に向けた要件整理。

## 2. 対象手法（調査対象）

- 既定候補
  - PaDiM / PatchCore / SimpleNet / EfficientAD / SAA+ / GLASS
- 追加候補（本調査で含める）
  - RD++、STFPM、RegAD、CFlow-AD、FastFlow、DRAEM、CutPaste、VisionAD、Diffusion系（AnoDDPM / DiffusionAD）、WinCLIP、PromptAD、SPADE、CFA/CFA+
- 再追加候補（本調査で含める）
  - RD4AD

    ```text
    5. SOTAs / 代表モデル
    現在、産業用画像異常検知でよく比較対象になる代表モデルは以下です：
      - PaDiM (Patch Distribution Modeling)
      - PatchCore
      - CFA / CFlow (Flow-based)
      - FastFlow
      - SimpleNet
      - EfficientAD
      - RD4AD
        これらは上記ベンチマークで一貫して評価されており、半導体外観検査でも応用されています。
    ```

## 3. 調査ソース

- 論文データベース：arXiv、IEEE Xplore、ACM DL、Elsevier、Springer、MDPI
- ベンチマーク・リーダーボード：MVTec AD / LOCO、VisA、Real-IAD、BTAD、DeepPCB
- 実装リポジトリ：GitHub / Papers With Code（公式/サードパーティ）
- 産業事例・技術ブログ・学会スライド：CVPR/ICCV/WACV/NeurIPS/AAAIワークショップ、企業Techブログ

## 4. 調査観点（テンプレート）

- 問題設定：教師あり/教師なし/ゼロショット、画像/パッチ/セグメンテーションの粒度
- モデル概要：特徴抽出（ResNet/ViT等）、メモリバンク/蒸留/フロー/再構成/拡散/CLIP系の区分
- データ要件：必要な良品/不良枚数、解像度、前処理（タイル、Retinex/CLAHEなど）
- 報告性能：主要指標（AUROC/AUPR/PRO/sPRO/mIoU/Dice）、データセットごとのスコアと順位
- 速度・資源：報告される推論速度・GPU/CPU条件、モデルサイズ、学習時間
- ロバスト性：照明・反射・視差、ドメインシフト、新種欠陥（OOD）への記述
- 再現性：公式実装有無、依存バージョン、再現報告、活発なIssue/PR状況
- ライセンス：OSS種別、商用条件、重みの再配布可否
- 導入要点：想定ライン構成、前処理・閾値決め、監視ポイント
- ギャップ：未検証領域、今後の実験課題、評価不足の観点

## 5. 調査方法

- スクリーニング
  - 直近3年（2023–2025）を主対象、引用の多い2020–2022の基盤手法も補足
  - タイトル・アブストラクトで一次選別、実装/スコア/適用事例の有無で二次選別
- エビデンス収集
  - 各手法テンプレートに沿って主要情報を抜粋し、表形式で比較可能化
  - ベンチマーク横断のスコアは原著/追試の両方を記録し、前処理差異を注記
- 品質管理
  - 二名レビュー（可能なら）で相互チェック、参照URLと日付を必ず付与
  - markdownlint/cSpellを通し、体裁・用語表記を統一

## 6. スケジュール（文献調査）

- Week 1: スコープ確定、キーワード設計、初回スクリーニング（30–50件）
- Week 2: 詳読・抜粋（優先20件）、比較表ドラフト作成
- Week 3: 追加探索（不足カテゴリ補完）、産業事例収集、ドラフト更新
- Week 4: レビュー/整形、採用候補案の提示、次段（実験計画）への要件整理

## 7. 成果物

- 調査レポート（Markdown）：各手法シート＋横断比較表（指標/資源/適用性/ライセンス）
- 参考文献リスト（Bib/URL付き）
- 採用候補ショートリスト（用途別：省データ立上げ／高速ライン／新種欠陥）
- 次段提案：実験設計ドラフト（データ要件・評価指標・優先アブレーション）

## 8. リスクと対策（文献調査に特有）

- 報告値の不整合：元論文の設定差（解像度/前処理/閾値）を注記し同条件比較を避ける
- リポジトリ非公開/リンク切れ：アーカイブミラーや代替実装を探索、著者連絡も検討
- 商用制約：ライセンス条項を早期確認し、候補から除外または注記
- 情報の新陳代謝：定期的な再検索（週次）でアップデートを反映

## 9. 優先深掘り（提案）

- 教師なし・高精度：RD++ / STFPM / RegAD
- 高速・省資源：FastFlow / CFlow-AD
- 微小・低コントラスト：DRAEM / Diffusion系
- 新種欠陥・省ラベル：VisionAD / WinCLIP / PromptAD
- 立上げ容易：CutPaste / PaDiM / PatchCore / EfficientAD
